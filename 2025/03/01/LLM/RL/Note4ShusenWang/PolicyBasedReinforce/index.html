Policy Based Reinforce

Policy Network π(a∣s;θ)\pi(a|s;\theta)π(a∣s;θ)

Policy Network: Use a neural net to approximate π(a∣s)\pi(a|s)π(a∣s).

  • Use policy network π(a∣s;θ)\pi(a|s;\mathbf{\theta})π(a∣s;θ) to approximate π(a∣s)\pi(a|s)π(a∣s).

  • θ\thetaθ: trainable parameters of the neural net.

  • Σa∈Aπ(a∣s;θ)=1\Sigma_{a\in A} \pi(a|s;\theta) = 1Σa∈A​π(a∣s;θ)=1

Policy-Based Reinforcement Learning

State-value function: how good the situation is in state sss

  • Vπ(st)=EA[Qπ(st,A)]=Σaπ(a∣st)⋅Qπ(st,a).V\pi(s_t) = \mathbb{E}_A[Q_\pi(s_t,A)] = \Sigma_a \pi(a|s_t) \cdot Q_\pi(s_t,a).Vπ(st​)=EA​[Qπ​(st​,A)]=Σa​π(a∣st​)⋅Qπ​(st​,a).

  • Vπ(st)=EA[Qπ(st,A)]=∫π(a∣st)⋅Qπ(st,a) da.V\pi(s_t) = \mathbb{E}_A[Q_\pi(s_t,A)] = \int \pi (a|s_t)\cdot Q_\pi(s_t,a) \,da.Vπ(st​)=EA​[Qπ​(st​,A)]=∫π(a∣st​)⋅Qπ​(st​,a)da.

Approximate state-value function:

  • Approximate policy function π(a∣st)\pi(a|s_t)π(a∣st​) by policy network π(a∣st;θ)\pi(a|s_t;\theta)π(a∣st​;θ).

  • Appproximate value function Vπ(st)V_\pi(s_t)Vπ​(st​) by: Vπ(st)=Σaπ(a∣st;θ)⋅Qπ(st,a).V\pi(s_t) = \Sigma_a \pi(a|s_t;\theta) \cdot Q_\pi(s_t,a).Vπ(st​)=Σa​π(a∣st​;θ)⋅Qπ​(st​,a).

V(s;θ)=Σaπ(a∣s;θ)⋅Qπ(s,a).V(s;\theta) = \Sigma_a \pi(a|s;\theta) \cdot Q_\pi(s,a). V(s;θ)=Σa​π(a∣s;θ)⋅Qπ​(s,a).

Policy-Based learning: Learn θ\thetaθ that maximizes J(θ)=ES[V(S;θ)]J(\theta) = \mathbb{E}_S[V(S;\theta)]J(θ)=ES​[V(S;θ)].

Policy gradient ascent

  • Observe state s.

  • Update policy by: θ←θ+β⋅∂V(s;θ)∂θ\theta \leftarrow \theta + \beta \cdot \frac{\partial V(s;\theta)}{\partial \theta}θ←θ+β⋅∂θ∂V(s;θ)​

Policy Gradient

∂V(s;θ)∂θ=Σa∂π(a∣s;θ)⋅Qπ(s,a)∂θ=Σa∂π(a∣s;θ)∂θ⋅Qπ(s,a)(Pretend Qπ is independent of θ)=Σaπ(a∣s;θ)⋅∂logπ(a∣s;θ)∂θ⋅Qπ(s,a)=EA[∂logπ(a∣s;θ)∂θ⋅Qπ(s,A)]\begin{align} \frac{\partial V(s;\theta)}{\partial \theta} &= \frac{ \Sigma_a \partial \pi(a|s;\theta) \cdot Q_\pi(s,a) }{\partial \theta} \\ &=\Sigma_a \frac{ \partial \pi(a|s;\theta) }{\partial \theta}\cdot Q_\pi(s,a) \quad (Pretend \ Q_\pi \ is \ independent \ of \ \theta)\\ &= \Sigma_a \pi(a|s;\theta) \cdot \frac{ \partial log \pi(a|s;\theta) }{\partial \theta} \cdot Q_\pi(s,a) \\ &= \mathbb{E}_A[\frac{ \partial log \pi(a|s;\theta) }{\partial \theta} \cdot Q_\pi(s,A) ] \end{align} ∂θ∂V(s;θ)​​=∂θΣa​∂π(a∣s;θ)⋅Qπ​(s,a)​=Σa​∂θ∂π(a∣s;θ)​⋅Qπ​(s,a)(Pretend Qπ​ is independent of θ)=Σa​π(a∣s;θ)⋅∂θ∂logπ(a∣s;θ)​⋅Qπ​(s,a)=EA​[∂θ∂logπ(a∣s;θ)​⋅Qπ​(s,A)]​​

Calculate Policy Gradient for Discrete Actions

  1. If the actions are discrete, e.g. action space A\mathcal{A}A = {“left”,“right”,“up”},…

    Use ∂V(s;θ)∂θ=Σa∂π(a∣s;θ)∂θ⋅Qπ(s,a)\frac{\partial V(s;\theta)}{\partial \theta} = \Sigma_a \frac{ \partial \pi(a|s;\theta) }{\partial \theta}\cdot Q_\pi(s,a)∂θ∂V(s;θ)​=Σa​∂θ∂π(a∣s;θ)​⋅Qπ​(s,a)

    1. Calculate $f(a,\theta) = \frac{ \partial \pi(a|s;\theta) }{\partial \theta}\cdot Q_\pi(s,a) $ for every action a∈Aa \in \mathcal{A}a∈A.

    2. Policy gradient: ∂V(s;θ)∂θ=f(left,θ)+f(right,θ)+f(up,θ)\frac{\partial V(s;\theta)}{\partial \theta} = f(left,\theta) + f(right,\theta) + f(up,\theta)∂θ∂V(s;θ)​=f(left,θ)+f(right,θ)+f(up,θ)

  2. If the actions are continuous, e.g. action space A\mathcal{A}A = [0,1].

    Use ∂V(s;θ)∂θ=EA∼π(⋅∣s;θ)[∂logπ(a∣s;θ)∂θ⋅Qπ(s,A)]\frac{\partial V(s;\theta)}{\partial \theta} = \mathbb{E}_{A\sim\pi(\cdot|s;\theta)}[\frac{ \partial log \pi(a|s;\theta) }{\partial \theta} \cdot Q_\pi(s,A) ]∂θ∂V(s;θ)​=EA∼π(⋅∣s;θ)​[∂θ∂logπ(a∣s;θ)​⋅Qπ​(s,A)] It’s difficult to calculate the ∫\int∫ of π\piπ. So we use Monte Carlo Approximation.

    1. Randomly sample an action a^\hat{a}a^ according to the PDF π(⋅∣s;θ)\pi(\cdot | s;\theta)π(⋅∣s;θ)

    2. Calculate g(a^,θ)=∂logπ(a^∣s;θ)∂θ⋅Qπ(s,a^)g(\hat{a},\theta) = \frac{ \partial log \pi(\hat{a}|s;\theta) }{\partial \theta} \cdot Q_\pi(s,\hat{a})g(a^,θ)=∂θ∂logπ(a^∣s;θ)​⋅Qπ​(s,a^).

    [!NOTE]

    Obviously, EA[g(A,θ)]=∂V(s;θ)∂θ\mathbb{E}_A[g(A,\theta)] = \frac{\partial V(s;\theta)}{\partial \theta}EA​[g(A,θ)]=∂θ∂V(s;θ)​.

    g(a^,θ)g(\hat{a},\theta)g(a^,θ) is an unbiased estimate of ∂V(s;θ)∂θ\frac{\partial V(s;\theta)}{\partial \theta}∂θ∂V(s;θ)​.

    1. Use g(a^,θ)g(\hat{a},\theta)g(a^,θ) as an approximation to the policy gradient ∂V(s;θ)∂θ\frac{\partial V(s;\theta)}{\partial \theta}∂θ∂V(s;θ)​. (This is called Monte Carlo approximation. The unbiased estimate is important.)

    [!IMPORTANT]

    This approach also works for discrete actions.

Algorithm

  1. Observe the state sts_tst​.

  2. Randomly sample action ata_tat​ according to π(⋅∣st;θt)\pi(\cdot | s_t;\theta_t)π(⋅∣st​;θt​)

  3. Compute qt≈Qπ(st,at)q_t \approx Q_\pi(s_t,a_t)qt​≈Qπ​(st​,at​) (some estimate).

  4. Differentiate policy network: dθ,t=∂log π(at∣st,θ)∂θ∣θ=θt\mathbf{d}_{\theta,t} = \frac{\partial log \ \pi(a_t | s_t, \theta)}{\partial \theta} |_{\theta = \theta_t}dθ,t​=∂θ∂log π(at​∣st​,θ)​∣θ=θt​​

  6. Update policy network: θt+1=θt+β⋅g(at,θt)\theta_{t+1} = \theta_t + \beta \cdot \mathbf{g}(a_t, \theta_t)θt+1​=θt​+β⋅g(at​,θt​).
